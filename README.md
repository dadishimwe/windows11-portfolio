# Windows 11 Portfolio — Dadi Ishimwe

Windows 11 inspired personal portfolio built with Next.js. A fork of [KasperiP/windows11-portfolio](https://github.com/KasperiP/windows11-portfolio), customized for [dadishimwe](https://github.com/dadishimwe).

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Configuration

Edit `config/site.ts` for your name, links, and branding. Set your live URL in `.env.local`:

```
NEXT_PUBLIC_SITE_URL=https://dadishimwe.com
NEXT_PUBLIC_BLOG_URL=https://blog.dadishimwe.com
```

## Deploy on Vercel

Push to GitHub and import the repo on [Vercel](https://vercel.com). No Cloudinary or Spotify env vars are required for a successful build.
